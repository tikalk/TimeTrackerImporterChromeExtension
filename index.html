Tie - Tikal's Time Tracker Sheet Importer

Tie is a a Chrome extension which will let you post your Montly excel time sheet to Tikal's time tracker.

UPDATE: 01/09/2012 - Chrome extenstion for Tie

The Excel Importer has been updated to follow the new API and instructions of developing extensiton for Chrome. This Google Chrome extension will load the bookmarklet script automatically once you enter the time tracker page.

Download and install Excel-Importer chrome extension

How To Install The Extension

  1. Open Google Chrome's extension page (url: chrome://chrome/extensions/)
  2. Drag the extension file you've downloaded and drop it into this page
  3. approve installation.

Requirements

Your excel should have 3 columns in the following order: day, start time, end time.
you can download an example of such an excel file.
